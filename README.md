# Simple ToDo App

Personal project to practice filter and API consumption.

## Practical concepts
- Fetching and posting data
- Conditional render
- Array Manipulation
- Contexts

## How to run

### Pre-requisites
- Node.js (v22)
- npm

### Installation
```bash
git clone https://github.com/felipe-u/simple-todo-app.git
cd simple-todo-app
npm install
```

### Run
```bash
npm run dev
```
